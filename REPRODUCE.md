# How to Reproduce This Project

Step-by-step guide to reproduce the intent classifier from scratch.
Total time: ~30-40 minutes. Cost: $0.

---

## Prerequisites

| What | Why | How to get |
|------|-----|------------|
| Google account | Access to Colab | You probably have one |
| Browser | Run Colab | Chrome/Firefox/Edge |
| Python 3.10+ | Run dataset generator locally | `python --version` |
| Git | Push to GitHub | `git --version` |

**No GPU, no paid services, no special hardware required.**

---

## Step 1: Generate Dataset (~5 min)

```bash
# Clone or create project directory
mkdir intent-classifier && cd intent-classifier

# Create seeds.py and generate_dataset.py (see project files)
# Then run:
python generate_dataset.py
```

Output: `dataset.csv` with ~2,877 labeled examples.

**Verify:**
```bash
wc -l dataset.csv          # Should be ~2878 (header + data)
head -5 dataset.csv         # Check format: text,label
```

**To customize intents:**
1. Edit `seeds.py` — add/modify seed examples for your classes
2. Edit `generate_dataset.py` — adjust templates and augmentation
3. Re-run `python generate_dataset.py`

---

## Step 2: Train Model in Colab (~15 min)

### 2.1 Open Colab
1. Go to [colab.research.google.com](https://colab.research.google.com/)
2. Upload `train.ipynb` (File → Upload notebook)
3. Upload `dataset.csv` to Colab's file panel (left sidebar → folder icon → drag & drop)

### 2.2 Select GPU
1. Runtime → Change runtime type → Hardware accelerator → T4 GPU
2. Click Save

### 2.3 Run Training
1. Runtime → Run all (Ctrl+F9)
2. Wait ~10 minutes for all cells to complete

### 2.4 Check Results
After cell 6 (Evaluation), you'll see:
- Classification report (precision, recall, F1 per class)
- Confusion matrix plot
- Overall accuracy

**Target:** F1 > 0.85 overall, > 0.90 for RAG class.

### 2.5 Download Results
The last cell auto-downloads:
- `intent-classifier-onnx.zip` — trained ONNX model
- `confusion_matrix.png` — evaluation visualization

## Step 4: Publish (~10 min)

### GitHub
```bash
git init
git add .
git commit -m "Intent classifier for RAG chatbot"
git remote add origin https://github.com/YOUR_USERNAME/intent-classifier.git
git push -u origin main
```

Note: if model.onnx.data > 100MB, either use Git LFS or host model on HuggingFace.

### HuggingFace Hub (optional)
1. Create account at [huggingface.co](https://huggingface.co/)
2. Uncomment and run cell 11 in the notebook
3. Your model will be at `huggingface.co/YOUR_USERNAME/intent-classifier-rubert-tiny2`

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Colab says "No GPU" | Try again later, or use CPU (slower but works) |
| `dataset.csv` not found in Colab | Upload via left sidebar file panel |
| ONNX export fails | Make sure `onnx` and `onnxruntime` are installed |
| Quantization fails | Use non-quantized model (larger but works identically) |
| Low accuracy (<80%) | Check dataset quality, increase epochs to 7-10 |
| `model.onnx.data` too large for GitHub | Use Git LFS: `git lfs track "*.onnx.data"` |

## Customization

### Different intents
1. Edit `LABELS` in train.ipynb: `["rag", "chat", "followup", "your_new_intent"]`
2. Add seeds and templates for new intent in seeds.py / generate_dataset.py
3. Regenerate dataset, retrain

### Different base model
Replace `MODEL_NAME` in train.ipynb:
- `ai-forever/ruBert-base` — better quality, 700MB, slower
- `bert-base-multilingual-cased` — multilingual, 700MB
- `distilbert-base-multilingual-cased` — multilingual, lighter

### Different language
Replace seeds with your language, use appropriate base model.
